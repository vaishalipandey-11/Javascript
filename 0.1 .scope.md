‚úÖ What is Scope?
Scope defines where variables can be accessed or used in your code.

In JS, variables are only available in the region (scope) in which they are defined.

üß± Types of Scope in JavaScript
| Type               | Description                                                                           |
| ------------------ | ------------------------------------------------------------------------------------- |
| **Global Scope**   | Variables defined outside any function or block ‚Äì available everywhere                |
| **Function Scope** | Variables declared inside a function ‚Äì only accessible in that function               |
| **Block Scope**    | Variables declared with `let`/`const` inside `{}` ‚Äì only accessible inside that block |
| **Lexical Scope**  | Inner functions can access variables from outer functions (closure behavior)          |


| **Scope Type**               | **Where Defined**                             | **Accessible Where?**                       | **Example Code**                                                                                     | **Output** / Notes                        |
| ---------------------------- | --------------------------------------------- | ------------------------------------------- | ---------------------------------------------------------------------------------------------------- | ----------------------------------------- |
| **Global Scope**             | Outside of all functions or blocks            | Everywhere in the file                      | `let a = 5;<br>function foo() { console.log(a); }<br>foo();`                                         | ‚úÖ Prints `5` inside function              |
| **Function Scope**           | Inside a function using `var`, `let`, `const` | Only inside that function                   | `function test() { let x = 10; console.log(x); }<br>test();<br>// console.log(x); ‚ùå`                 | ‚ùå ReferenceError outside function         |
| **Block Scope**              | Inside `{}` using `let` / `const` only        | Only inside that `{}` block                 | `if (true) { let y = 20; }<br>// console.log(y); ‚ùå`                                                  | ‚ùå Block-scoped: not accessible outside    |
| **`var` (not block scoped)** | Inside block using `var`                      | Available outside block (inside function)   | `if (true) { var z = 30; }<br>console.log(z); ‚úÖ`                                                     | ‚úÖ Prints `30`, not block-scoped           |
| **Lexical Scope**            | Function inside function (closure)            | Inner function has access to outer vars     | `function outer() { let msg = "hi"; function inner() { console.log(msg); } inner(); }<br>outer();`   | ‚úÖ Inner accesses outer `msg`              |
| **Scope Chain**              | Nested functions with variable lookup         | Looks up parent scopes if not found locally | `let a = 1;<br>function one() { let b = 2;<br>function two() { console.log(a, b); } two(); } one();` | ‚úÖ Prints `1 2` ‚Äî looks up the scope chain |
